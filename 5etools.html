FAQ

5etools 有百科嗎？ 這裡（英文）。 歡迎貢獻內容。

5etools 免費嗎？ 是的！此外，你還可以在 get.5e.tools 下載原始碼。檢視 自述檔案 README 或者百科以獲取幫助。

我可以為 5etools 英文站點貢獻內容嗎？ 原始碼的拷貝在 GitHub 上公開。注意，這是一個映象，所以並不保證內容始終最新。

我可以為 5etools 中文站點（正在檢視的此站點）貢獻內容嗎？ 基本上這個網站只為私下使用，所以沒有人力維護，也沒空審核PR。

瀏覽器與裝置支援？ 最新的桌面版 Chrome/Edge Chromium/Firefox。

怎麼管理我自己的數據？ 並沒有賬號系統一說。一切都儲存在 cookies 當中。如果你清除這些 cookies，或者關閉你的隱私模式視窗，儲存的內容就會永久消失。爲了安全儲存你的數據，你可以通過設定菜單下載你自己的數據。

還有別的我該知道的東西嗎？ 在滑鼠懸浮時，按住 SHIFT 以就地固定住一個視窗。所有內容都有工具提示，而你作為 DM 將永恒。
